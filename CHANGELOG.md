# Changelog

## v0.4.16 — Interface and Launch Update

- Убраны устаревшие скриншоты интерфейса.
- Обновлены Windows-скрипты запуска.
- Упрощены README и VPS-инструкция.
- Локальные данные входа перенесены в `.env.local.example`.

## v0.4.5 — Upload Validation

- Исправлена обработка заблокированных расширений при загрузке.
- Если пользователь пытается загрузить `.exe`, `.bat`, `.cmd` и другие опасные типы, приложение показывает обычную ошибку.

## v0.4.1 — Security Update

- Добавлена CSRF-защита для браузерных POST-форм.
- Добавлены security headers.
- Настроены secure cookies и trusted hosts.
- Добавлена проверка слабых секретов при production-запуске.
- Добавлена возможность использовать хеш пароля администратора.
- Усилена проверка паролей пользователей.
- Добавлены ограничения загрузки и блокировка опасных расширений.
- Dockerfile переведён на запуск приложения не от root.
- Добавлены тесты безопасности.

## v0.4.0 — Accounts and Storage

- Добавлены регистрация и вход пользователей.
- Добавлен раздел “Мои загрузки”.
- Загрузки привязываются к пользователю после входа.
- Добавлены журналы действий, скачиваний и очистки.
- Добавлена поддержка MinIO/S3.
- Добавлен Docker Compose с приложением, PostgreSQL и MinIO.
- Добавлены настройки лимитов загрузки и входа.
- Добавлены тесты и CI.

## v0.3.0 — Backend Base

- Добавлена подготовка под PostgreSQL.
- Добавлены Alembic-миграции.
- Добавлены Docker-файлы.
- Добавлены health/status endpoints.
- Добавлены журналы скачиваний и очистки.
